# Overview

**Project Name:** Handcrafted Haven

**Group Members:** Julyanne Lee (Team Lead and all roles)

Handcrafted Haven is a web-based marketplace for artisans and crafters to showcase and sell unique handcrafted items. The platform connects creators with customers who value creativity, authenticity, and sustainable consumption.

# Purpose

- Provide a pltform for artisans to display and sell their products
- Enable customers to browse, search, and review handcrafted items
- Foster a community of creators and conscious consumers

# Project Scope & Features

- Responsive web application built with Next.js and Node.js
- Seller profiles and product listings
- Search, filter, reviews, and ratings
- Accessible design (WCAG 2.1 Level AA) and consistent branding

# Tools and Technology Stack

- Front-End: HTML, CSS, JavaScript, Next.js
- Back-End: Node.js, a database
- Project Management: GitHub Boards
- Code Management: Git and GitHub Repository
- Deployment / Cloud Platform: Vercel
